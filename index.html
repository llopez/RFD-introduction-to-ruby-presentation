Introducción a Ruby

¿Qué es ruby?

Es un lenguaje de programación poderoso y flexible que puedes usar en el desarrollo aplicaciones desktop, web o simplemente de consola.

Es un lenguaje de alto nivel, interpretado y orientado a objetos.

¿Qué es ruby?

  • De alto nivel, lo que quiere decir que leer y escribir Ruby es realmente fácil; ¡se parece mucho al inglés común y corriente!
  • Interpretado porque no necesitas un compilador para ejecutar Ruby. Puedes escribirlo en tu propio computador
  • Orientado a los objetos porque les permite a los usuarios manipular estructuras de datos llamadas objetos, para construir y ejecutar programas. En Ruby, todo es un objeto.

Todo en Ruby es un objeto

          1.class                 # => Fixnum
          1.5.class               # => Float
          “hello”.class           # => String
          true.class              # => TrueClass
          false.class             # => FalseClass
          nil.class               # => NilClass
          99999999999999999.class # => Bignum

Cadena de caracteres

Una cadena de caracteres es una secuencia de caracteres entre comillas. Este objeto pertenece a la clase String. 

          “hello world”.class      # => String
          “hello world”.length     # => 11
          “hello world”.capitalize # => “Hello world”
          “hello world”.upcase     # => “HELLO WORLD”
          “hello world”[0]         # => “h”

Array

Un arreglo es un objeto donde es posible almacenar una serie de otros objetos. Los arreglos pertenecen a la clase de Array.

          array = [1, 2, 3, 4, 5]
          array[0] # => 1
          array[1] # => 2
          array[2] # => 3
          array[3] # => 4
          array[4] # => 5
          
          for i in array
            puts i
          end
          
          # OR
          
          array.each{|i| puts i}

Hash

Un hash es como un arreglo en el cual se puede acceder a los valores por una clave.

          hash = {first_name: ‘Luis’, last_name: ‘lopez’, age: 29}
          hash[:first_name] # => ‘Luis’
          hash[:last_name]  # => ‘Lopez’
          hash[:age]        # => 29

Método

Un método es como una función que puede recibir o no parámetros

"El return esta implícito"

Ejemplo:

          def say_hello(name)
            return "Hello guys, My name is #{name}"
          end
          
          # OR
          
          def say_hello(name)
            "Hello guys, My name is #{name}"
          end
          
          say_hello('Luis') # => "Hello guys, My name is Luis"

Clase

Una clase se puede definir como una plantilla para crear un objeto. Esta constituida por atributos y métodos:

Podemos pensar a los atributos como las características que definen a un objeto y a los métodos como aquello que define el comportamiento del objeto.

Clase

Ejemplo

          class Person
            attr_accessor :first_name, :last_name, :age

            def initialize(first_name, last_name, age)
              @first_name = first_name
              @last_name = last_name
              @age = age
            end

            def name
              “#{first_name.capitalize} #{last_name.capitalize}”
            end

            def say_hello
              puts “Hello guys, My name is #{name}”
            end
          end

Instancia

Una instancia de un objeto es la creación del mismo partiendo de una clase.

Dada nuetra clase Person, podemos instanciar un objeto de la siguiente manera:

          person = Person.new(‘luis’, ‘lopez’, 29)
          person.first_name # => ‘luis’
          person.last_name  # => ‘lopez’
          person.age        # => 29
          person.name       # => ‘Luis Lopez’
          person.say_hello  # => ‘Hello guys, My name is Luis Lopez’

Herencia

La herencia es la relación entre una clase general y otra más específica, padre e hija.

Nos facilita la reutilización y la extensibilidad del código.

          class Worker < Person
            attr_accessor :job 
          end
          
          worker = Worker.new(‘Joaquín’, ‘Vicente’, 30)
          worker.first_name   # => ‘Joaquín’
          worker.last_name    # => ‘Vicente’
          worker.age          # => 30
          worker.job          # => nil
          worker.job = ‘Programmer’
          worker.job          # => ‘Programmer’
          worker.say_hello    # => ‘Hello guys, My name is Joaquín Vicente’
